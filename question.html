Question-1: What are the different ways to select an element in the DOM?
There is 5 ways to select an element in the dom.they are - 1. getElementById() for single element selection, 2. querySelector() for with CSS selectors, 3. querySelectorAll() when selecting multiple elements, 4. getElementsByClassName() ,5.getElementsByTagName() for all the element with same classname and tag name.
Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML is used adding or modifying HTML content dynamically. innerText is used when we want only visible text and respect formatting. textContent is used when you want all text, including hidden text, and better performance.
Question-3: What is event delegation in the DOM?
Event delegation is a smart way to handle events on multiple elements without adding event listeners to each one.we attach one event listener to a parent, and it listens for events from its child elements. One event listener handles all buttons and Works for new buttons added later.
